1. Download and Install `Node.js` and `Ganache`
   
2. Download `MetaMask` Extension
   
3. Install `truffle` globally
   
    ```bash
    npm install -g truffle
    ```
    

4. Install dependencies
   
    ```bash 
    npm install
    ```

5. Open `Ganache` and create a local blockchain with `"New Workspace"` option. 

6. Add `Ganache` as a new network and add `Ganache users' wallet` to `MetaMask` and use the platform.

7. Compile the Smart Contracts

    ```bash
    truffle compile
    ```

8. Migrate the Smart Contract to the Blockchain

    ```bash
    truffle migrate
    ```

9. Run Node.js Server

    ```bash
    node server.js
    ```
